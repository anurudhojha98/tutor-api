# Tutor-Bin apis
# Steps to run apis
1. Clone the repo
2. Run npm install
3. Run npm start
4. Import postman collection in the postman https://www.getpostman.com/collections/dec641edc1a0b197f13b 
5. Use the apis
6. I have also added swagger example demo you can find it by http://localhost:3200/api-docs/
